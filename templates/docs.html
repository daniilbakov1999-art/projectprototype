Паттерн проектирования

Что такое Prototype?

Prototype (Прототип) - это порождающий паттерн проектирования, который позволяет копировать объекты, не вдаваясь в подробности их реализации. Вместо создания объекта с нуля через конструктор, мы запрашиваем клон у уже существующего объекта-прототипа.

Backend (Python / Flask)

Реализация на сервере

В файле server.py инициализируется класс ShapeRegistry, который хранит базовые прототипы круга, прямоугольника и треугольника. Метод клонирования принимает входящие параметры размеров и цвета с фронтенда, создавая независимый клон с уникальным UUID.

Frontend (JS / HTML5)

Интерактивное взаимодействие

Фронтенд отслеживает клики по кнопкам добавления фигур, считывает текущие значения из полей цвета и размера, и отправляет POST-запрос на API бэкенда. При нажатии на кнопку «Дублировать выбранное», на сервер улетает слепок текущего состояния выделенной фигуры для создания точной рантайм-копии.
